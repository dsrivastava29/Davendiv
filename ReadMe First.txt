Following project is made for academic purpose. 
It contains hands on for Web Tools course and related concepts.

-Spring MVC Framework (using Annotation)
-Hibernate (JPA 2.0) [Criteria, Projections, HQL]
-MySQL
-Spring Web Flow Integration (for admin section)

Development tools used: Eclipse Spring Tool Suite (STS) and Maven
(Consider Maven update if project shows any error after loading on eclipse)

: Pojo's have been created based on E-R model and modified for the use in application.

It is still under development but basic features work fine.

No need to run schema still uploading here as its last great running copy of schema.
